## 프로젝트 개요

**산불 위험·경보 펄스맵**은 공공데이터 기반의 **산불위험예보 정보**와 **긴급재난문자 정보**를 수집·저장·가공하여,  
지역별 산불 위험도와 실제 경보 발생 현황을 **지도 위에서 한눈에 확인할 수 있도록 시각화하는 서비스**입니다.

이 프로젝트는 단순히 재난문자를 나열하는 것이 아니라,  
**“위험도가 높은 지역”**과 **“실제로 재난문자가 발송된 지역”**을 함께 보여줌으로써  
사용자가 현재 상황과 잠재적 위험을 동시에 파악할 수 있도록 설계했습니다.

즉, 이 서비스는 **예방형 정보(위험 예보)** 와 **실제 경보 정보(재난문자)** 를 결합한  
**공공안전형 펄스맵 서비스**를 목표로 합니다.

### 개발 목적
- 공공데이터를 활용해 **산불 위험 지역을 직관적으로 시각화**
- 산불위험예보와 긴급재난문자를 결합해 **신뢰도 높은 정보 제공**
- 수집 → 저장 → 가공 → 시각화까지 이어지는 **데이터 기반 서비스 구조 구현**
- 향후 침수, 대기질, 산사태 등 다른 주제로 확장 가능한 **펄스맵 플랫폼 기반 마련**

### 핵심 기능
- 산불위험예보 API 수집
- 긴급재난문자 API 수집
- 화재/산불 관련 키워드 필터링
- 위험도 데이터 및 재난문자 DB 저장
- 지역별 위험도와 실제 경보 데이터를 지도 시각화
- 최근 경보 이력 및 지역별 상세 정보 제공
